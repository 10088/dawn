# Dawn

Dawn 取「黎明、破晓」之意， 是一个轻量的「任务管理和构建」工具。

- [English](README.md)

## 特点

- 采用中间件技术，封装常用功能，易于扩展，方便重用
- 支持 pipeline 让多个 task 协同完成构建任务
- 简单、一致的命令行接口，易于开发人员使用
- 支持基于「中心服务」管理中件间和工程模板
- 支持搭建私有中心服务，并统一下发构建规则，易于团队统一管理
